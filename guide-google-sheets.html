🗓️ Guide de mise en place

Gérer vos tarifs et réservations via Google Sheets — sans jamais toucher à GitHub

1

Créer le Google Sheet

Allez sur sheets.google.com et créez une nouvelle feuille. Nommez-la "Les Terrasses d'Annecy — Gestion".

Ce fichier contiendra deux onglets : Tarifs et Reservations.

2

Créer l'onglet "Tarifs"

Renommez le premier onglet en Tarifs (double-clic sur l'onglet en bas). Saisissez exactement ce tableau :

Tarifs
Reservations
ABCDE
1 Studio Basse Moyenne Haute Festival
2 Semnoz 390 460 540 620
3 Lanfonnet 390 460 540 620
4 Parmelan 380 450 530 620
⚠️ Les noms de studios en colonne A doivent être exactement : Semnoz, Lanfonnet, Parmelan (avec majuscule, sans accent).
3

Créer l'onglet "Reservations"

Créez un deuxième onglet (bouton + en bas) et nommez-le Reservations (sans accent). Saisissez ce tableau :

Tarifs
Reservations
ABC
1 Studio Date Statut
2 Semnoz 2026-06-07 réservé
3 Exemple à remplacer…
📅 La date en colonne B est toujours un samedi, au format AAAA-MM-JJ (ex : 2026-07-12). C'est la date d'arrivée.

Pour ajouter une réservation : ajoutez simplement une nouvelle ligne. Pour la supprimer : effacez la ligne.

4

Publier le fichier sur le web

C'est l'étape clé. Elle permet au site de lire vos données. Faites-la une seule fois.

📊 Les Terrasses d'Annecy — Gestion
📄 Fichier
Nouveau
Ouvrir
Importer
→ Partager > Publier sur le web…

Dans la fenêtre qui s'ouvre :

  • ✓ Sélectionnez "Tout le document"
  • ✓ Format : "Valeurs séparées par des virgules (.csv)"
  • ✓ Cliquez "Publier"
  • ✓ Confirmez en cliquant "OK"
⚠️ Ne partagez pas l'URL qui s'affiche — elle ne sera pas utile. On utilisera une URL différente à l'étape suivante.
5

Récupérer l'identifiant du fichier

Regardez l'URL de votre Google Sheet dans la barre d'adresse du navigateur :

https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit

La partie surlignée en jaune est votre identifiant de fichier. Copiez-la.

💡 C'est une longue suite de lettres et chiffres entre les deux /d/ et /edit.
6

Mettre l'identifiant dans le fichier config.js

Sur GitHub, ouvrez le fichier config.js (qui sera créé par votre développeur). Vous verrez cette ligne :

const SHEET_ID = 'COLLER_VOTRE_ID_ICI';

Remplacez COLLER_VOTRE_ID_ICI par votre identifiant copié à l'étape 5, puis cliquez Commit changes.

✅ C'est la seule manipulation GitHub de tout ce guide. Après ça, tout se passe dans Google Sheets.
✓

Utilisation quotidienne — C'est tout !

Une fois la mise en place terminée, voici tout ce que vous avez à faire pour gérer votre site :

Modifier un tarif : Ouvrez Google Sheets → onglet Tarifs → changez le chiffre → Entrée. C'est immédiatement pris en compte sur le site.

Ajouter une réservation : Onglet Reservations → nouvelle ligne → saisissez le studio, la date du samedi d'arrivée, "réservé" → Entrée.

Supprimer une réservation : Effacez simplement la ligne correspondante.

💡 Vous pouvez modifier le Google Sheet depuis votre téléphone, une tablette, n'importe quel ordinateur. Le site se met à jour automatiquement à chaque visite d'un visiteur.

🎉 Mise en place terminée !

Le site Les Terrasses d'Annecy lit maintenant ses données depuis Google Sheets.

Plus jamais besoin de toucher à GitHub pour gérer les tarifs et les disponibilités.
